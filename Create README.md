# Car Price Prediction Project

## Project Overview
This project predicts car selling prices using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn
- Streamlit

## Dataset
Car Price Dataset

## Models Used
- Linear Regression
- Decision Tree
- Random Forest

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Deployment

## Run Project

pip install -r requirements.txt
